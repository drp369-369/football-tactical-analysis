# Football Tactical Analysis using Event Data

This repository contains football analytics projects focused on extracting
actionable tactical insights from real match data.

## Project 1: Space Utilization and Passing Structure
- Match: France vs Croatia (2018 FIFA World Cup Final)
- Analysis of team shape, spatial occupation, and passing networks
- Focus on tactical interpretation and decision-making support

## Key Skills Demonstrated
- Football event data analysis
- Spatial visualization and network analysis
- Tactical interpretation and communication
- Reproducible analytical workflows

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, NetworkX
